# WanderQuest

WanderQuest is a full-stack web application designed to help users explore, plan, and manage trips in an interactive way.
It provides user authentication, trip management, journals, and engaging frontend pages connected to a backend API.

--

## Tech Stack
- **Frontend**
  - React.js
  - HTML5
  - CSS3
  - Java Script
 
- **Backend**
   - Node.js
   - Express.js
   - MongoDB (with Mongoose)

--

## Features

- User registration and login
- Create, view, and manage trips
- Travel journal entries
- Interactive frontend pages
- RESTful API for frontendâ€“backend communication

--

## How To Run

- **Run the Backend**
   cd backend
   npm install
   npm start

   It runs on:
   **http://localhost:5000**

- **Run the Frontend**
   cd frontend
   npm install
   npm start

   It runs on:
   **http://localhost:3000**
